# Sample Markdown

A **basic** but _real_ Markdown preview. No network, no scripts.

## Features

- Headings
- **Bold** and _italic_
- `inline code`
- Lists (like this one)

1. First
2. Second
3. Third

### Code block

```js
function hello(name) {
  return "hi " + name; // rendered as escaped text
}
```

### Links

- [Safe link](https://example.com)
- [Dangerous link](javascript:alert(1)) is stripped to plain text.
